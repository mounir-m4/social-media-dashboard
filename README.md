### Run the project

To run this project simply install node dependencies and run sass, see below:

```shell
   npm install 
```

add to command in your package.json

```shell
"sass": "sass --watch scss/main.scss:dist/css/main.css"
```

then run a this command to watch sass

```shell
   npm run sass
```
